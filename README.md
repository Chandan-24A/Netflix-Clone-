# Netflix UI Clone

A simple **Netflix landing page UI clone** built using only **HTML and CSS**.  
This project focuses on improving frontend fundamentals and understanding how real-world website layouts are designed.

## Features

- Netflix-inspired homepage design  
- Responsive layout  
- Clean and modern UI  
- Sections included:
  - Navigation bar  
  - Hero section  
  - Content sections with images & video  
  - FAQ section  
  - Footer  


## Tech Stack

- HTML5  
- CSS3  


## Learning Outcomes

This project helped me to:

- Understand HTML tags and structure more clearly  
- Learn real-world UI layout techniques  
- Work with modern CSS properties  
- Use Flexbox and Grid effectively  
- Implement responsive design using media queries  
- Improve overall frontend fundamentals

A Netflix-style landing page with a hero banner, content sections, and FAQ layout.
